Editor:
- various types of editors for specific purpose
- enum class with types
- one show function which takes type of editor table for argument
- Toggle function to show and hide editor
- Color and shaders editor
- Noise parameters editor

EditorTree:
- Ieditor - abstract class of varius types of editors
- EditorManager - setting editors activity, updating and rendering them. 
Also using ImGui initializing functions Newframe() EndFrame() etc.

Mesh loader:
- ability to load own meshes of a objects
- applying objects to the map
